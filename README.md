# random-images
随机图片，此项目实例为二次元随机壁纸，用来做背景图API使用。
# 预览地址
https://api.aynu.top/ACG
# 使用教程
1.上传文件到网站目录即可，img.txt 文本是随机函数所需要的图片地址，我这里选的是自己上传到微博图床的图片，可以是任意的图片地址。  
2.使用自己的域名的话需要将 api.css 里面的 https://api.aynu.top/ACG/api.php 改为你的域名的api.php文件真实路径
